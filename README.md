# Sistem_Web_Perpustakaan_Digital-_Sekolah
Sistem Perpustakaan berbasis web yang dibuat menggunakan PHP dan MySQL untuk mengelola perpustakaan sekolah secara digital. Fitur meliputi katalog buku, manajemen buku dan anggota, peminjaman dan pengembalian, log aktivitas, pengaturan, berita dan pengumuman. Sistem dirancang secara kreatif oleh pembuat dan implementasi kode dibantu teknologi AI.
